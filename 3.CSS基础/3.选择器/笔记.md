# 选择器

选择器: 帮助你精准的选中想要的元素

## 简单选择器

1. ID选择器
2. 元素选择器
3. 类选择器
4. 通配符选择器

*, 选中所有元素

5. 属性选择器

根据属性名和属性值选中元素

6. 伪类选择器

选中某些元素的某种状态
(尽量按照1-4的顺序声明这4个伪类, 不然会出现问题,后续会讲解)
1) link: 超链接未访问时的状态

2) visited: 超链接访问过后的状态

3) hover: 鼠标悬停状态

4) active: 激活状态, 鼠标按下状态

爱恨法则: love hate

7. 伪元素选择器

生成并选中某个元素内部的第一个子元素或最后一个子元素

before

after

## 选择器的组合

1. 并且
2. 后代元素:     空格
3. 子元素:         >
4. 相邻兄弟元素:   +
5. 兄弟元素:       ~
